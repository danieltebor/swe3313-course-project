# swe3313-course-project

## Introducing Hank's Mineral Emporium
Welcome to the SWE 3313 project repository for team NullPointerException! All project artifacts, presentations, documentation, and source code are located here. Hank from Breaking Bad needs to sell his collection of rare minerals and has commissioned a small website, Hank's Mineral Emporium. The website will implemented using C#, ASPNET Core, and JSON.
